## Installation

To install the `Substract CLI`, ensure you have Node.js version 20.0.0 or later.

```bash
npm install -g ocr-cli
```

Or

```bash
npx ocr-cli
```

Or

````bash
bunx ocr-cli
```

## Requirements

- **Node.js v23.0.0+**
````
